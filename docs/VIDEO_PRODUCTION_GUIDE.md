## Production Guidelines

### Equipment Requirements

**Minimum Equipment:**
- Computer with screen recording software
- Microphone (USB or built-in)
- Screen resolution: 1920x1080
- Recording software: OBS, Camtasia, or similar

**Recommended Equipment:**
- External microphone (USB or XLR)
- Pop filter
- Audio recording software
- Video editing software
- Graphics tablet (for annotations)

### Recording Settings

**Video Settings:**
- Resolution: 1920x1080 (1080p)
- Frame rate: 30 fps
- Codec: H.264
- Bitrate: 5-8 Mbps

**Audio Settings:**
- Sample rate: 44.1 kHz or 48 kHz
- Bit depth: 16-bit or 24-bit
- Format: WAV or high-quality MP3
- Bitrate: 128-192 kbps (MP3)

### Recording Best Practices

**Before Recording:**
- Close unnecessary applications
- Disable notifications
- Test microphone levels
- Practice the script
- Prepare the screen

**During Recording:**
- Speak clearly and at a moderate pace
- Use consistent microphone distance
- Avoid mouse movements when not needed
- Highlight important elements
- Pause between sections

**After Recording:**
- Review the recording
- Check audio quality
- Note sections to re-record
- Organize files properly

### Editing Guidelines

**Video Editing:**
- Remove mistakes and pauses
- Add smooth transitions
- Include zoom highlights
- Add text overlays for key points
- Maintain consistent pacing

**Audio Editing:**
- Remove background noise
- Normalize audio levels
- Add background music (subtle)
- Ensure clear speech
- Add captions if possible

**Graphics & Branding:**
- Use consistent branding
- Include KMainCMS logo
- Use brand colors
- Add intro/outro sequences
- Include chapter markers

## Troubleshooting

### Audio Issues
- **Problem:** Low volume
- **Solution:** Check microphone levels, boost in post-production

- **Problem:** Background noise
- **Solution:** Use noise reduction, record in quieter environment

### Video Issues
- **Problem:** Blurry screen
- **Solution:** Record at higher resolution, use sharper scaling

- **Problem:** Laggy recording
- **Solution:** Close other apps, use hardware acceleration

### Editing Issues
- **Problem:** Out of sync
- **Solution:** Align audio and video tracks, adjust timing

- **Problem:** Export errors
- **Solution:** Check file format, reduce file size, use different codec
