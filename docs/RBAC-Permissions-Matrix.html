🎭 KMainCMS RBAC Permissions Matrix

Complete overview of all modules, functions, and role permissions in the system

📊 Module Overview

🏠 Dashboard

  • View - Access main dashboard overview and statistics

👥 Members Management

  • View - View member directory and profiles
  • Create - Add new members to the system
  • Edit - Update member information and details
  • Delete - Remove members from the system
  • Export - Export member data to external formats

🏢 Departments Management

  • View - View department list and information
  • Create - Create new departments
  • Edit - Update department details
  • Delete - Remove departments
  • Manage - Full department management including assignments

🖼️ Gallery Management

  • View Public - View public church photos (all users)
  • View All - View all photos including pending (managers)
  • Request Upload - Submit photos for approval (all logged-in users)
  • Upload - Upload photos directly (approved) (managers)
  • Edit - Edit photo metadata and album details
  • Delete - Remove photos and albums
  • Approve - Approve/reject pending uploads (managers)
  • Manage - Full gallery management and organization

📁 Documents Management

  • View - View document library and files
  • Upload - Upload new documents
  • Edit - Update document metadata
  • Delete - Remove documents
  • Manage - Full document management and organization

💰 Treasury Management

  • View - View financial overview and summaries
  • Manage - Full treasury management
  • Reports - Access financial reports and statements
  • Transactions - View and manage financial transactions
  • Budgets - Create and manage budgets

📱 SMS Management

  • View - View SMS history and logs
  • Send - Send SMS messages to individuals or groups
  • Manage - Full SMS management
  • Templates - Create and manage SMS templates
  • Campaigns - Create and manage SMS campaigns

📢 Announcements Management

  • View - View announcements and notices
  • Create - Create new announcements
  • Edit - Update existing announcements
  • Delete - Remove announcements
  • Publish - Publish announcements to various channels

✅ Approvals Management

  • View - View approval requests and status
  • Request - Submit items for approval
  • Approve - Approve submitted requests
  • Reject - Reject submitted requests
  • Manage - Full approval workflow management

🔐 Users Management

  • View - View user accounts and profiles
  • Create - Create new user accounts
  • Edit - Update user information
  • Delete - Remove user accounts
  • Manage Roles - Assign and manage user roles

⚙️ Settings Management

  • View - View system settings
  • Edit - Update system settings
  • Manage - Full settings management

📅 Events Management

  • View - View events and calendar
  • Create - Create new events
  • Edit - Update event details
  • Delete - Remove events
  • Manage - Full event management

💳 Payments Management

  • View - View payment history and transactions
  • Process - Process payments and transactions
  • Refund - Process refunds
  • Manage - Full payment management

📊 Reports Management

  • View - View available reports
  • Generate - Generate custom reports
  • Export - Export reports to various formats

📝 Content Management

  • View - View content and pages
  • Create - Create new content
  • Edit - Update existing content
  • Delete - Remove content
  • Publish - Publish content to live site

📈 Analytics

  • View - View basic analytics and statistics
  • Advanced - Access advanced analytics and insights

🔒 Security Management

  • View - View security logs and settings
  • Manage - Manage security settings
  • Audit - Access security audit logs

📱 Telegram Integration

  • View - View Telegram integration status
  • Manage - Manage Telegram bot and settings
  • Broadcast - Send broadcast messages via Telegram

📲 Mobile App Management

  • View - View mobile app status
  • Manage - Manage mobile app settings and content

🖥️ System Monitoring

  • View - View system metrics and performance
  • Manage - Manage monitoring settings and alerts

🔍 SEO Management

  • View - View SEO settings and metadata
  • Manage - Manage SEO configuration

♿ Accessibility Management

  • View - View accessibility settings
  • Manage - Manage accessibility features

🧪 Testing Tools

  • View - View testing tools
  • Execute - Run tests and diagnostics

📚 Documentation

  • View - View system documentation
  • Edit - Update documentation
